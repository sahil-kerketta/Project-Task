GitHub (/ˈɡɪthʌb/ ⓘ) is a proprietary developer platform that allows developers to create, store, manage, and share their code. It uses Git to provide distributed version control and GitHub itself provides access control, bug tracking, software feature requests, task management, continuous integration, and wikis for every project.[9] GitHub, headquartered in San Francisco, is operated by Github, Inc., a subsidiary of Microsoft since 2018.[10]
